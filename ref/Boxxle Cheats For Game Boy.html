    Boxxle Cheats For Game Boy

    1. View Credits

      Hold the A Button, B Button, and Up on the control pad while at the Title Screen.

      Contributed by: Land of GameCubes 

    2. Complete any puzzle to win

      If you enter this password, you will be able to select from any of the puzzles on the last stage. All of them will appear to be completed. Complete any one of them and you will see the ending.

      Effect Effect
      Q, [spade], [spade], N All puzzles completed

      Contributed by: IntyLab
